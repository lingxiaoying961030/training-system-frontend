1️⃣ 项目卡片封面

设计原则：intentionality, not intensity — 克制比花哨更高级
两种方案对比。核心思路：保持安静，用场景背景营造氛围，不堆砌动态元素。

方案 A：场景背景 + 底部灌木 + 图标

用已有的 daybg/nightbg/meadow 做背景，一层灌木做前景层次，图标站在地面上。白天场景有一朵极慢飘动的云（30s 周期），其余安静。

图像标注基础

白天 — 一朵慢云

NLP 文本分类

夜晚 — 纯静态

语音标注进阶

草地纹理 — 纯静态

方案 B：纹理渐变 + 角落小装饰

保持现在的渐变色底，叠一层极淡的地板纹理（6% 透明度），角落放一个低透明度的小装饰。比场景版更安静。

图像标注基础

绿色 + 角落树影

NLP 文本分类

蓝色 + 角落 Junimo

语音标注进阶

暖色 + 角落小鸡

3️⃣ 动态交互：hover 时才活

平时安静，鼠标移上去时微微响应 — 图标上浮，装饰物跳一下
不做持续动画。把"活"的感觉留给用户主动触发的瞬间。试试 hover 下面的卡片👇

6️⃣ 登录页：安静但有温度

极慢的云（40s 一朵） + 静态树影 + 角落一只安静的小动物 + Junimo 微呼吸
不巡逻、不蹦跳。场景是"清晨刚醒来的牧场"——安静，但有生命存在。
